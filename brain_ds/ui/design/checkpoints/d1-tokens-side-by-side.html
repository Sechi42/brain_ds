D.1 Token Review — modern-design-tokens

Locked: --bg-main #09090b · --bg-panel #121214 · --border-subtle rgba(63,63,70,0.4) · --radius-md 8px · --fw-{regular,medium,semibold}
What to inspect
  • Carbon/zinc backgrounds — NO absolute black (#000), surfaces should read as #09090b / #121214.
  • Refined borders — rgba(63,63,70,0.4) on panels, no white-tinted borders.
  • Radius scale — buttons/tabs use --radius-ui (8px via --radius-md). NOTE: bumped from 6px (design catalog) to 8px (spec/DIV-4) — corners should look slightly softer than before. Not a regression.
  • Font weights — labels should use --fw-medium / --fw-semibold, not size jumps.
  • Motion — hover/focus respects prefers-reduced-motion.
  • Out of scope (D.2/D.3): segmented controls, mini-sidebar, 4-zone layout.
